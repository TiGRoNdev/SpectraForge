### 4. Локальные Git Hooks ✅

#### Созданы скрипты установки hooks
- **`scripts/setup_git_hooks.sh`** (Linux/macOS)
- **`scripts/setup_git_hooks.bat`** (Windows)

#### Реализованные hooks:
- **pre-commit**: Проверка форматирования и безопасности
- **pre-push**: Проверка качества и сборки
- **commit-msg**: Валидация формата commit message (Conventional Commits)

#### Возможности hooks:
```bash
# Pre-commit проверки
- Форматирование staged файлов
- Проверка небезопасных функций
- Валидация namespace'ов
- Проверка TODO/FIXME с номерами issue

# Pre-push проверки  
- Запуск quality_check.sh
- Проверка сборки проекта
- Базовая валидация

# Commit-msg проверки
- Conventional Commits формат
- Правильные типы коммитов
```
