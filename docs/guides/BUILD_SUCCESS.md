## 🐛 Известные ограничения

Текущая версия (v1.0) имеет следующие ограничения:

1. **Растеризация** - упрощенная проекция вокселей (не полный Gaussian Splatting)
2. **Motion Vectors** - заглушка (нулевые векторы)
3. **Апскейлинг** - только Bilinear (DLSS/FSR2 - заглушки)
4. **Текстуры** - загружаются но не используются в рендере
5. **SH коэффициенты** - упрощенная инициализация

Эти ограничения документированы для будущих улучшений. См. [FREQVOX_SPONZA_IMPLEMENTATION.md](FREQVOX_SPONZA_IMPLEMENTATION.md) для деталей.

## 🎓 Научная ценность

Демо демонстрирует инновационный подход к рендерингу:

- **Frequency-Domain Shading** - использование DCT/FFT в real-time
- **Foveated Rendering** - адаптивное распределение GPU ресурсов
- **Temporal Coherence** - использование информации предыдущих кадров
- **Neural Integration** - применение AI для апскейлинга
