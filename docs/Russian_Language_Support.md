# Поддержка русского языка в Engine4D

## Обзор

Engine4D полностью поддерживает русский язык в GUI системе, включая корректное отображение кириллических символов, эмодзи и специальных символов.

## Особенности русскоязычной поддержки

### ✅ Поддерживаемые функции

1. **Текстовые элементы**
   - Кнопки с русским текстом
   - Текстовые метки (UILabel)
   - Поля ввода текста
   - Подсказки (placeholder text)

2. **Кодировка**
   - UTF-8 поддержка на всех платформах
   - Правильная настройка Windows консоли
   - Эмодзи и специальные символы

3. **Интерфейс**
   - Локализованные сообщения
   - Русские подписи элементов
   - Инструкции на русском языке

### 🛠️ Техническая реализация

#### Windows
```cpp
// Настройка UTF-8 для консоли
SetConsoleCP(CP_UTF8);
SetConsoleOutputCP(CP_UTF8);

// Поддержка виртуального терминала для эмодзи
HANDLE hOut = GetStdHandle(STD_OUTPUT_HANDLE);
DWORD dwMode = 0;
if (GetConsoleMode(hOut, &dwMode)) {
    dwMode |= ENABLE_VIRTUAL_TERMINAL_PROCESSING;
    SetConsoleMode(hOut, dwMode);
}
```

#### Универсальная локаль
```cpp
// Установка системной локали
try {
    std::locale::global(std::locale(""));
} catch (const std::exception& e) {
    // Обработка ошибок локализации
}
```

## Настройка проекта для русского языка

### 1. Включение заголовков
```cpp
#include <locale>
#include <codecvt>

#ifdef _WIN32
#include <Windows.h>
#include <io.h>
#include <fcntl.h>
#endif
```

### 2. Инициализация локали
```cpp
void setupRussianLocale() {
#ifdef _WIN32
    SetConsoleCP(CP_UTF8);
    SetConsoleOutputCP(CP_UTF8);
#endif
    
    try {
        std::locale::global(std::locale(""));
    } catch (...) {
        // Fallback к системной локали
    }
}
```

### 3. Использование в main()
```cpp
int main() {
    setupRussianLocale();
    
    std::cout << "🌟 Добро пожаловать!" << std::endl;
    // ... остальной код
}
```

## Рекомендации по разработке

### ✅ Лучшие практики

1. **Кодировка файлов**
   - Сохраняйте .cpp файлы в UTF-8
   - Используйте UTF-8 BOM при необходимости

2. **Строковые литералы**
   - Используйте обычные строки: `"Русский текст"`
   - Избегайте wide strings если не требуется

3. **Консольный вывод**
   - Всегда настраивайте локаль в начале программы
   - Используйте std::cout для UTF-8 текста

4. **Тестирование**
   - Тестируйте на Windows и Linux
   - Проверяйте отображение в разных терминалах

### ❌ Чего избегать

1. **Смешанные кодировки**
   - Не используйте Windows-1251
   - Избегайте ANSI кодировок

2. **Хардкод локали**
   - Не фиксируйте конкретную локаль
   - Используйте системную локаль как fallback

3. **Игнорирование ошибок**
   - Обрабатывайте исключения локализации
   - Предусмотрите запасные варианты

## Отладка проблем с русским языком

### Распространенные проблемы

1. **Кракозябры в консоли Windows**
   ```bash
   # Решение: настройка UTF-8
   chcp 65001
   ```

2. **Неправильные символы в GUI**
   - Проверьте кодировку исходного файла
   - Убедитесь в правильной настройке локали

3. **Отсутствие эмодзи**
   - Включите поддержку виртуального терминала
   - Обновите шрифт терминала

### Диагностика

```cpp
// Проверка текущей локали
std::cout << "Локаль: " << std::locale().name() << std::endl;

// Тест UTF-8
std::cout << "Тест: Привет мир! 🌍" << std::endl;
```
