## 🔧 Technical Details

### **P0.3 Architecture Changes:**

#### **Before (God Class):**
```cpp
class Engine {
private:
    // 20+ responsibilities mixed together:
    InputState inputState_;
    std::chrono::steady_clock::time_point lastFrameTime_;
    float deltaTime_;
    glm::vec3 cameraPos;
    glm::vec3 cameraFront;
    // ... и т.д.
};
```

#### **After (Component Composition):**
```cpp
class Engine {
private:
    // Clear separation of concerns:
    std::unique_ptr<Core::GameLoopManager> gameLoop_;
    std::unique_ptr<Core::WindowManager> windowManager_;
    std::unique_ptr<Core::InputManager> inputManager_;
    std::unique_ptr<Core::SceneCoordinator> sceneCoordinator_;
};
```
